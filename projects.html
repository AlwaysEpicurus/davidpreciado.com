Project Overview: Personal Website for David Preciado

This site is more than a digital business card — it’s a storytelling-driven experience designed to reflect clarity, trust, and momentum, the very values I bring to my work. Built with a focus on clean aesthetics, intuitive flow, and technical craftsmanship, the site showcases both my capabilities and personality in a way that’s accessible, responsive, and modern.

Phase 1 Details (Completed)

  • Deployed the site on Oracle Cloud Object Storage.
  • Secured the site using Cloudflare for DNS, SSL/TLS, and WAF protection.
  • Implemented version control via GitHub for code collaboration and management.
  • Integrated Bootstrap for responsive, mobile-friendly design.
  • Used Jira for task planning and iterative delivery.

Phase 2 Details (Completed)

  • Refined the user experience with a storytelling-driven approach.
  • Polished content to ensure clarity and engagement.
  • Enhanced visual design with custom CSS for a modern aesthetic.
  • Standardized layout, scroll arrows, and viewport behavior across pages .
  • Refined section spacing, structure, and responsive design polish
