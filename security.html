Security Measures

Committed to safeguarding davidpreciado.com with industry best practices

Protecting the integrity, confidentiality, and availability is a top priority. I implement modern security best practices and regularly review our infrastructure to keep up my skills.

DNS and SSL/TLS Security

DNS Management

Traffic is routed securely using Cloudflare, protecting against DDoS attacks and boosting site performance.

SSL/TLS Configuration

Full (Strict) mode ensures traffic encryption for maximum security between users and servers.

HSTS Enforcement

HTTP Strict Transport Security (HSTS) enforces HTTPS to prevent downgrade attacks.

Web Application Firewall

Traffic Filtering

Blocks malicious traffic and protects against threats like SQL injection and cross-site scripting (XSS).

JavaScript Protection

Cloudflare’s JS Challenge reduces unauthorized access risks to sensitive scripts.

Additional Security Measures

Secure Cloud Hosting

Public access on Oracle Cloud is restricted to only essential resources.

Regular Updates

Security configurations and dependencies are reviewed regularly for vulnerabilities.

Access Control

Only authorized administrators have backend access.
